# DB Visualizer

An AI-powered database visualization platform that provides intelligent insights through natural language queries. Built with a Flask backend and Next.js frontend.

## Repository Structure

```
DB-Visualizer/
├── backend/     # Python Flask API with AI agents and BigQuery integration
└── frontend/    # Next.js 14 web application with TypeScript and Tailwind CSS
```

## Tech Stack

- **Backend**: Python, Flask, LangGraph, OpenAI GPT-4, Google BigQuery
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, Zustand
- **Database**: PostgreSQL
